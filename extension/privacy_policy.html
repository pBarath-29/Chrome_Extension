ToS Clarity — Privacy Policy

Last updated: May 2026

ToS Clarity does not collect, store, or sell any personal data. No account is required.

What data is processed

When you click "Review Agreement", the extension:

  • Extracts the visible text of the current page
  • Sends the page URL and extracted text to the backend server running on your device

Where your data goes

The backend server (running at 127.0.0.1:8000 by default) forwards the document text and URL to the Groq API for analysis using a Llama model. Groq's data handling is governed by the Groq Privacy Policy and Terms of Use.

What is NOT collected

  • No browsing history is recorded
  • No analysis results are stored by the extension
  • No data is sent to any server other than your local backend and the Gemini API
  • No cookies, tracking pixels, or analytics are used

Local backend

The backend server runs entirely on your device. It does not persist any document text or analysis results to disk. Data exists only in memory for the duration of a single analysis request.

Contact

For questions or concerns, open an issue on the project's GitHub repository.
